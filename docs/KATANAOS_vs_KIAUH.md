# KATANAOS vs. KIAUH

## Kern-Features

| Feature | KIAUH | KATANAOS |
|---------|:-----:|:--------:|
| Klipper / Moonraker / Mainsail / Fluidd | ✅ | ✅ |
| Firmware bauen (menuconfig) | ✅ | ✅ |
| Firmware flashen | manuell | **auto-detect** |
| Updaten | ✅ | ✅ auto-branch |
| Uninstall | ✅ | ✅ |

## Nur KATANAOS

- USB Auto-Detection (`lsusb` → DFU, Serial, Klipper)
- Engine Switch (Klipper ↔ Kalico ↔ RatOS)
- Crowsnest + KlipperScreen
- Smart Probes (Beacon, Carto, Eddy)
- Backup & Restore (ZIP + Git)
- Profile (minimal/standard/power)
- Status-Dashboard im Menü
- Diagnose & Logs
- PolicyKit Auto-Setup
- Linux Host MCU automatisch

## KIAUH voraus

- Stabilität (jahrelang getestet)
- Community + Tutorials
- Dokumentation
